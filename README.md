# Music Match

This web app lets you compete with your friends to see who knows each others music taste the best!

## Requirements

- NodeJS v18.16.0

## How To Use

1. Running the Server

   - `cd server`
   - `npm run dev`

2. Running the Client

   - `cd client`
   - `npm install`
   - `npm start`

3. [Launch the Client Local Browser](http://localhost:3000/)
